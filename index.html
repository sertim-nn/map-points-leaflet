Инструкция по работе с MapPoints

1. Импорт точек из Excel

Перейдите на вкладку «Импорт» и загрузите Excel-файл (.xlsx или .xls) со столбцами:

  • Группа — название группы
  • Имя — название точки
  • ШИРОТА (Широта) — координата широты
  • ДОЛГОТА (Долгота) — координата долготы
  • Число — значение (@, $, &, ?, N, X или число)
  • Цвет — HEX (#a8c5e2) или два через запятую для градиента 70%/30%
  • Форма — circle, drop или flag
  • Адрес — текстовый адрес
  • Код ТТ — код торговой точки

2. Управление группами

На вкладке «Карта» вы можете:

  • Включать/выключать отображение групп чекбоксами
  • Показать/скрыть все группы кнопками
  • Выделить границы группы кнопкой с рамкой

3. Мои списки

Создавайте списки точек, выделяя области на карте:

  • Нажмите «Добавить список»
  • Выберите форму: прямоугольник, круг или многоугольник
  • Зажмите ЛКМ и выделите область
  • Экспортируйте в Excel кнопкой экспорта

4. Зоны доставки

Загружайте зоны из TXT или JSON на вкладке «Импорт». Затем используйте раздел «Обновление расписания» для загрузки Excel с изменениями.

  • Дни доставки — текущие дни (Пн,Ср,Пт)
  • Цикл — чет / нечет (или пусто = каждую неделю)
  • Дни доставки new — новые дни (если изменились)
  • Цикл new — новая цикличность
  • Цвет — HEX-цвет подсветки (#f59e0b)

5. Бекап данных

На вкладке «Ещё»:

  • Бекап — скачивает JSON со всеми данными
  • Восстановить — загружает данные из бекапа
  • Очистить — удаляет все данные
Совет: Данные автоматически сохраняются в браузере.
